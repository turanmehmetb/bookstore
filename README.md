## Description

Simple Bookstore API

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov

# load test (requires loadtest package -> npm install -g loadtest)
$ npm run loadtest
```
## Swagger

Swagger UI can be accessed from http://localhost:3000/api
